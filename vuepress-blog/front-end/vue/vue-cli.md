Runtime-Compiler 和 Runtime-only

- 如何在开发中使用template，选择Runtime-Compiler。
- 如何在开发中使用的是.vue文件夹开发，那么可以选择Runtime-only

```javascript
// 一下两者等价
// 这种情况需要编译器
new Vue({
  el: '#app',
  components: { App },
  template: '<App>'
})

// 这种情况不需要
new Vue({
  el: '#app',
  render: h => h(App)
})
```
